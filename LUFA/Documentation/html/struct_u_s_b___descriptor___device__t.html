uint8_t USB_Descriptor_Device_t::SerialNumStrIndex

String index for the product's globally unique hexadecimal serial number, in uppercase Unicode ASCII.

Note:
On some AVR models, there is an embedded serial number in the chip which can be used for the device serial number. To use this serial number, set this to USE_INTERNAL_SERIAL. On unsupported devices, this will evaluate to 0 and will cause the host to generate a pseudo-unique value for the device upon insertion.
